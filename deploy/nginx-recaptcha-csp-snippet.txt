# Фрагмент для nginx: если reCAPTCHA не грузится, часто виноват заголовок Content-Security-Policy.
# Подставьте домены в свою политику (одна строка add_header Content-Security-Policy ... или map).
#
# В script-src добавьте:
#   https://www.recaptcha.net https://www.gstatic.com https://www.google.com
#
# В frame-src добавьте (значок / виджет):
#   https://www.google.com https://recaptcha.google.com https://www.recaptcha.net
#
# Пример только добавок (остальная политика у вас своя):
#   script-src ... https://www.recaptcha.net https://www.gstatic.com https://www.google.com;
#   frame-src ... https://www.google.com https://recaptcha.google.com https://www.recaptcha.net;
